# Secure Access and IT Support at Qatar Foundation

## What This Project Is About

This project summarizes my IT Assistant work supporting secure access, resident identity verification, account permissions, and daily technical operations across Qatar Foundation housing facilities.

The work combined IT support with security-focused access control, because the environment required both smooth user service and protection against unauthorized access.

## What I Worked On

- Supported IT and access-control systems across residential facilities
- Helped verify resident identities and prevent unauthorized access
- Troubleshot hardware, software, and network issues
- Managed user accounts, permissions, and access-related support requests
- Communicated with residents, technical teams, and security staff
- Helped maintain secure and smooth daily operations

## Skills Practiced

- Access control
- Identity verification
- User account support
- Permission management
- Technical troubleshooting
- Security-aware communication
- Operational risk awareness

## Cybersecurity Connection

This work connects to cybersecurity areas such as IAM, GRC, SOC support, and security operations because it required controlling who had access, solving technical issues quickly, and protecting people and systems in a real environment.

## What I Learned

I learned that trust and security depend on consistent processes. Even small access-control mistakes can create risk, so permissions, identities, and technical issues must be handled carefully and documented clearly.
