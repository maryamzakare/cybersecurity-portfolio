# My Digital Forensics Work

## What This Project Is About

This project page summarizes my digital forensics skills and coursework, including how to examine digital evidence, preserve findings, and understand what happened on a system.

Digital forensics is important because security teams need to investigate incidents without destroying evidence.

## Why I Made It

I wanted to understand how investigators analyze computers, files, storage, and system artifacts after a security incident.

This connects to incident response because evidence must be collected and reviewed carefully before conclusions are made.

## What I Worked On

- Digital evidence analysis
- File and system artifact review
- OS forensics concepts
- Evidence preservation
- Timeline thinking
- Incident investigation support
- Documentation of findings

## Tools and Concepts

- Autopsy
- WinHex
- File system analysis
- Disk and hard drive evidence
- Metadata
- Deleted file concepts
- Chain of custody
- Evidence integrity
- Hashing concepts

## Cybersecurity Connection

Digital forensics supports cybersecurity by helping answer questions like:

- What happened?
- When did it happen?
- Which user or system was involved?
- What files were accessed or changed?
- Was data deleted, copied, or modified?
- What evidence supports the investigation?

## Skills I Practiced

- Careful investigation
- Evidence handling
- Technical documentation
- Attention to detail
- Security analysis
- Incident response support

## Privacy Note

This page does not include private evidence, real user data, confidential files, or screenshots that expose sensitive information.

Any examples shared publicly should be from labs, class exercises, or approved sample evidence.

## What I Learned

I learned that digital forensics requires patience, accuracy, and discipline. It is not enough to find something suspicious; the investigator must preserve evidence and explain findings clearly.

This skill is valuable for incident response, SOC analysis, threat investigation, and forensic analyst roles.
