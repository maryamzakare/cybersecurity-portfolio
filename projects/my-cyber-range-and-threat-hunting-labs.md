# My Cyber Range and Threat Hunting Labs

## What This Project Is About

This project summarizes hands-on cyber range labs I completed using SimSpace and related security training environments.

The labs focused on threat hunting, SOC operations, malware indicators, Windows files, scripting, Nmap, and incident response thinking.

## Incident Response Concepts

These labs helped me understand how security teams investigate suspicious activity and decide what action to take.

I also participated in class discussions and debate about different incident response strategies, including:

- Contain and recover
- Observe and monitor
- Protect and prevent
- Document and improve

The main lesson was that response decisions depend on the severity of the incident, business risk, available evidence, and whether the attacker is still active.

## Cybersecurity Connection

This project connects to real cybersecurity work because SOC analysts and incident responders need to search logs, identify indicators of compromise, analyze suspicious systems, and explain findings clearly.

## What I Learned

I learned that threat hunting is not only about finding alerts. It requires asking good questions, understanding normal behavior, searching for evidence, and connecting small indicators into a bigger picture.

These labs helped me build confidence with security tools and incident response workflows.
