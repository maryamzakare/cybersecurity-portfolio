# DDoS Attack Detection Research

## What This Project Is About

This project focused on DDoS attack detection using cybersecurity research papers, machine learning ideas, and CICIDS network traffic datasets.

I studied existing research about DDoS attacks and then worked on additional implementation ideas as a graduate-level cybersecurity project.

## Why I Made It

DDoS attacks can disrupt websites, services, networks, and business operations. Detecting DDoS traffic is important because defenders need to separate normal network behavior from attack traffic quickly.

This project helped me connect academic research with practical cybersecurity analysis.

## What I Worked On

- Researched DDoS attack detection papers
- Studied normal traffic vs attack traffic
- Reviewed existing research methods
- Presented DDoS research findings
- Worked with CICIDS network traffic data
- Added additional implementation ideas beyond the original paper
- Compared machine learning approaches for attack detection

## Dataset

I worked with CICIDS-style network traffic data, which includes network flow features that can be used to study benign traffic and attack traffic.

Examples of useful traffic features may include:

- Flow duration
- Packet counts
- Byte counts
- Source and destination behavior
- Protocol information
- Traffic rate
- Timing patterns
- Labels for benign or attack traffic

## Machine Learning Implementations

As part of the graduate-level work, I explored additional implementations such as:

- Logistic Regression
- Multi-Layer Perceptron
- Other possible classification models

The goal was to evaluate how different models could help classify benign and malicious network traffic.

## Cybersecurity Connection

This project connects directly to SOC analysis, threat detection, and network defense.

Security teams need to understand how attack traffic differs from normal traffic so they can detect DDoS activity, respond faster, and protect critical services.

## What I Learned

I learned that DDoS detection is not only about seeing high traffic volume. It also requires understanding network behavior, traffic features, model performance, and the limits of machine learning.

This project helped me understand how research can be extended by testing additional models and comparing different detection approaches.
