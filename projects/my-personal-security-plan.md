# My Personal Security Plan

## What This Project Is About

This project was about creating a full personal security plan for protecting devices, accounts, hard drives, personal data, and daily digital activity.

I looked at how someone can secure their computer, storage, online accounts, authentication methods, backups, and long-term security habits.

## Why I Made It

I wanted to understand how cybersecurity applies to real life, not only companies. A person can lose important files, accounts, identity, or privacy if they do not have a clear security plan.

This project helped me think like a security analyst: identify risks, choose controls, and create a realistic roadmap.

## What I Worked On

- Device security
- Hard drive and data protection
- Multi-factor authentication
- Password and account security
- Backup and recovery planning
- Security roadmap planning
- Risk prioritization

## Skills I Practiced

- Security planning
- Risk analysis
- MFA decision-making
- Documentation
- Personal data protection
- Security awareness

## Tools and Concepts

- Multi-factor authentication
- Device encryption
- Backups
- Password managers
- Account recovery
- Security checklists
- Privacy settings

## What I Learned

I learned that security is not only about tools. It is also about habits, planning, and making safer choices before something bad happens.

This project helped me understand how to explain cybersecurity in a practical way that normal users can follow.
